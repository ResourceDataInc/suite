Boundless SDK
=============

Boundless SDK provides tools for building web mapping applications backed by 
OpenGeo Suite.

To see usage, please run the suite-sdk script inside the bin directory.

The most current documentation on how to use this script is available at:
  http://suite.opengeo.org/opengeo-docs/apps/clientsdk.html

API documentation for Boundless SDK is found at:
  http://suite.opengeo.org/opengeo-docs/sdk-api/
