System requirements
-------------------

The following Windows versions are supported:

* Windows Vista
* Windows 7
* Windows 8
* Windows 10
* Windows Server 2008
* Windows Server 2012

OpenGeo Suite has the following system requirements:

* Memory: 1GB minimum (2GB recommended)
* Disk space: 1.2GB minimum (plus extra space for data)
* Browser: Internet Explorer, Firefox, Chrome
* Permissions: Administrative rights
* Software: .NET Framework 4.5 (`Download <https://www.microsoft.com/en-us/download/details.aspx?id=30653>`_)
